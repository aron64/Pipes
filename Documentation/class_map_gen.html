static int [,] MapGen.GenerateMap ( )
inlinestatic

Egy csőtérképet generáló algoritmus, fallal a szélén bejárat, kijárat és random csövek, legalább 1 útvonallal 0 = ellenőrizetlen 1 = fal 2 = egyenes 3 = kanyar 4 = meghatározatlan 8 = bejárat (mindig x = 0, y = 2) 9 = kijárat In the final version there are only 1,2,3,8,9

Visszatérési érték
int[MapSize+2, MapSize+2] map - csovek típussal, ki- és bejárat
