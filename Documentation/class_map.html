Detailed Description

Map osztály A játékteret definiáló objektumokat, Illetve a játékteret előkészítő függvényeket, metódusokat tartalmazza.

NEM ez az osztály fogja kezelni a csöveket.

Publikus komponensek: static int[] MapSize - OptionsMenu script változtatja static int[] mapShit - a balfelső sarok kordinátái egy Cube méreteit beszámítva Publikus fv-k, metódusok: void CreateEmptyMap - elhelyezi az elkerítést, beleértve a csapot és a kijáratot
